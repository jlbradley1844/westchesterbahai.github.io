This is Baha'i website of the Westchester Cluster. All documents hosted on this site are public. This
is generated by Hugo (Blowfish theme) under Github Pages via GitHub Actions.

## For quick changes
* Navigate to the document you wish to edit.
* Click the "pencil" icon to make edits, click again when done.
* Submit to master. This will create a "pull request" that admins can clean up and review for posting.

## Full Local Installation (Advanced Users Only!!!)
* Fork the repository
* Go to settings and set Github Pages source as master.
* Your clone should be ready and navigable under the published link.
* If you are proposing significant changes, you should push to a branch. (If you don't know what a "push" is, this
workflow probably doesn't apply to you.)

## Appendix: Deployment Gotchas (Hugo + GitHub Pages)

After migrating from Jekyll to Hugo, three issues caused 404 errors:

1. **GitHub Pages source must be set to "GitHub Actions"** (Settings → Pages → Source). The old "Deploy from a branch" mode does not work with the Hugo Actions workflow.
2. **Workflow only triggers on `master`** — changes on other branches do not deploy until merged.
3. **Custom domain gets wiped when switching the Pages source.** Re-enter `westchesterbahais.org` in Settings → Pages, wait for the green DNS check, then re-run the workflow manually. The site will not serve correctly at the custom domain until a fresh build runs after the domain is confirmed.
